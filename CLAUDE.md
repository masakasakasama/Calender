# CLAUDE.md

開発ガイド・エージェント間コーディネーション・不変条件（自動削除しない / 個人カレンダーを
取り込まない 等）と、進行中のリファクタ依頼は **[AGENTS.md](./AGENTS.md)** に集約しています。
作業前に必ず AGENTS.md を読んでください。

要点だけ:
- 予定を**自動削除しない**（softDelete はユーザーの明示削除だけ。重複は表示側で隠すのみ）。
- Google 取り込みは**共有カレンダー(@group) の ID 一致のみ**。個人カレンダーは取り込まない。
- 変更後は `npm run build` 緑、`appVersion` を上げる、main push で自動デプロイ。
- **Google 同期レイヤの一本化リファクタは Codex 担当**（AGENTS.md のタスク参照）。Claude は同期系ファイルを触らない。
